# Electron-learning
Learning demo for electron and nodejs



### 安装环境 
1. Mac 上安装 youtube-dl , 运行 ```brew install youtube-dl ```

2. 进入项目目录后, 运行 ```npm install```
3. 然后运行 ```npm install -g @vue/cli```
4. 启动程序, 运行 ```npm run electron:serve```  开启程序.
